# manghe
交友盲盒-抽对象-H5裂变分销系统全套源代码-情人节七夕同城单身交友

后台系统基于微擎、接入需要准备好 已认证微信服务号和微信支付.

系统支持分销[摆摊赚钱]模式、最多可以配置 3级、返点也能自己设置、自定义支持挺全面

系统要求 PHP7.1 + MySQL5.7，服务器环境部署使用 Centos7.9 + 宝塔面板配置较为方便、

挺有趣的系统，就是部署配置稍微麻烦。
